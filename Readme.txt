Проект запускается на двух разных портах: 3000 и 4221.
Каждый их них запускает свою фронтенд и бэкенд часть: 3000 отвечает за админ панель, а 4221 - за панель пользователя.
Обмен между портами происходит посредством файла items.json.

Для запуска админ панели необходимо выполнить:
cd Admin
node server.js

Для запуска пользовательской панели необходимо выполнить:
cd User
node server.js
